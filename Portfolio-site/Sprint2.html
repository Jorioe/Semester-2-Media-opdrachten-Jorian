Sprint 2

McMusic App

Voor sprint 2 van de McDonalds opdracht was ons idee om iets in de McDonalds app te maken dat de menu kiest aan de hand van je meest geluisterde muziekgenre. Daarvoor heb ik deze 2 schermen gemaakt. Bij de 1e iteratie heb ik een scherm gemaakt waarbij de meest geluisterde genre Romantiek is. Een voorbeeld van een gerecht dat daarbij zou kunnen passen was de veggie homestyle crispy chicken. Bij de 2e iterastie heb ik een scherm gemaakt waarbij de meest geluisterde genre Hip-Hop is. Hierbij heb ik het Big Mac voordelmenu gezet.

Iteratie 2

Spotify API

Om het idee ook echt te laten werken ben ik aan de slag gegaan met de Spotify API in .js. Voor mij was dit nog onbekend terrein, maar na een paar dagen tutorials volgen kon ik steeds een beetje meer. Hierdoor heb ik iets kunnen maken waarmee je alle afspeellijsten met de inbegrepen nummers van een gebruiker kan verkrijgen. Voor de demonstratie heb ik een paar random afspeellijsten opgeslagen zodat die worden getoond.
